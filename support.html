Open an issue on GitHub →

What to Include in Your Report

To help us resolve your issue as quickly as possible, please include the following details when contacting support or filing a bug report:

  • Your browser name and version (e.g., Chrome 130, Firefox 128, Safari 18)
  • Your operating system (e.g., Windows 11, macOS 15, Ubuntu 24.04)
  • A clear description of the issue or unexpected behavior
  • Steps to reproduce the problem (what you did before the issue appeared)
  • Screenshots or screen recordings, if possible
  • Whether you are using the dashboard locally, via start.mykk.us, or through the Chrome extension
