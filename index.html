Frequently Asked Questions

Got questions? We've got answers

Is MyKK really free?

Yes! The dashboard is open source and completely free. All core features — search, bookmarks, calendar, notepad, paint canvas, drag & drop, themes, cloud sync, and more — are included at no cost. Pro adds premium integrations like live weather, iFrame embeds, and task management.

What is the dashboard exactly?

It's a single HTML file (~400 KB) that you download and open in your browser. Set it as your homepage or new tab page. No server, no Docker, no terminal — just a file. It's also open source on GitHub.

Do I need the Chrome extension?

No. The dashboard works as a standalone HTML file. The extension works with the Cloud Dashboard at start.mykk.us, makes it easier to set as your start page, and manages your Pro subscription.

How do I activate Pro?

After subscribing, install the MyKK Chrome extension (if you haven't already), click the extension icon, and sign in with the same Google account you used to purchase. Pro features activate automatically.

Is there a refund policy?

Yes, we offer a 30-day money-back guarantee. If you're not satisfied with Pro, contact us at support@mykk.us for a full refund.

Does it collect any data?

The dashboard itself does not collect, store, or transmit any personal data — all settings are stored locally or synced via your own Google account. The mykk.us website uses Google Analytics for anonymous usage statistics. The code is open source so you can verify this yourself.
