Getting Started

MyKK is a single HTML file dashboard that replaces your browser's new tab or start page. There are no build tools, no server requirements, and no dependencies. Simply download the file and open it in your browser to get started.

Installation

There are three ways to start using MyKK:

  • Option 1: Download from GitHub — Download index.html from the GitHub repository and open it directly in your browser.
  • Option 2: Live Demo — Visit start.mykk.us to use the hosted version instantly.
  • Option 3: Chrome Extension — Install the Chrome extension to replace your new tab page with MyKK.

Setting as Homepage

  • Chrome: Go to Settings > On startup > Open a specific page > Add the local file path or start.mykk.us.
  • Firefox: Go to Settings > Home > Custom URLs and enter the file path or start.mykk.us.
  • Edge: Similar to Chrome — go to Settings > On startup > Open a specific page and add your URL.

Widgets

  • Search Bar: Supports 7 search engines — Google, Bing, DuckDuckGo, Brave, Yahoo, Ecosia, and Startpage. Click the engine icon to switch between them.
  • Favorites: Drag-and-drop bookmarks with auto-detected favicons. Three icon style options are available. Click the + button to add a new bookmark.
  • Calendar: A monthly calendar view with navigation arrows to move between months.
  • Clock & Date: Supports 12-hour and 24-hour formats with configurable seconds display and multiple date styles.
  • Notepad: A simple text editor that auto-saves to your browser. Copy contents or download as a text file.
  • Paint Canvas: Draw with brush, shapes, and text tools. Includes an eraser. Export your artwork as PNG or PDF.
  • Webcam: A mirror view with countdown timer and flash effect for quick selfies.

Customization

  • Click the gear icon to open Settings.
  • Themes: Choose from built-in gradient themes or set a custom background image or color.
  • Layout: Drag sections by their handle to reorder them. Resize widget columns to fit your preference.
  • Widget visibility: Toggle individual widgets on or off in Settings to show only what you need.

Cloud Sync

  • Click "Sign in with Google" in Settings to enable cloud sync.
  • Syncs your bookmarks, layout, theme, and widget settings across devices.
  • Uses Firebase — your data is stored in your Google account.
  • Works across all devices where you sign in with the same account.
  • To disable sync, sign out in Settings. Your data will revert to local storage.

Pro Features

  • Live Weather & Radar: Current conditions, 5-day forecast, and an interactive radar map with animated overlays.
  • iFrame Widgets: Embed Google Calendar, RSS feeds, X/Twitter timelines, or any URL directly in your dashboard.
  • Donetick Integration: Connect to your Donetick task manager to view and manage tasks.
  • Twitter/X Feed: Embed your timeline directly in the dashboard.

How to activate: Subscribe at mykk.us, install the Chrome extension, and sign in with the same Google account.

Self-Hosting

  • Download index.html from the GitHub repository.
  • Host on any web server, CDN, or simply open it locally as a file.
  • Compatible with Nginx, Apache, Cloudflare Pages, GitHub Pages, Netlify, and more.
  • No build step, no dependencies, no database required.
  • All data is stored in the visitor's browser localStorage.

Chrome Extension

  • Install from the Chrome Web Store.
  • Automatically sets MyKK as your new tab page.
  • Manages your Pro subscription — sign in with Google to activate.
  • The extension is optional. The dashboard works fully as a standalone HTML file.
